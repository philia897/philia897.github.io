《联邦学习》读书笔记

2022-03-24 ML ML, 联邦学习, 读书笔记 0 Comments Word Count: 1.4k(words) Read Count: 4(minutes)

《联邦学习》是杨强、刘洋等人编著的面向数据安全和隐私保护的机器学习的综述性的书，整体上不是很具体，但是能对联邦学习有一个比较大体的了解

作者：
杨强
刘洋
程勇
康焱
陈天健
于涵

1.引言

介绍人工智能的挑战以及联邦学习的分类和发展

1.1 人工智能面临的挑战

  1. 数据量庞大难以达到，通常得到的数据规模小，信息不全（需要打标）难以桥接的数据孤岛
  2. 数据所有权，用户隐私和数据安全的关注度不断提高，法律法规，在不同组织间收集数据和分享数据将变得越来越难，某些高度敏感的数据也不能无限制计算和使用。
  3. 如何在条条框框更多，解决数据碎片化和数据隔离的问题
  4. 将数据联合起来协同训练会导致数据失去掌控，整合后带来的性能赠一不好分配，分配效果不透明，加剧数据碎片化和孤岛分布
  5. 物联网和边缘计算，大数据分布在许多方面。eg：自动驾驶汽车在本地要处理信息，并且和其他的汽车和计算中心协同工作 =》 如何安全有效的实现模型在多个地点间的更新和共享

1.2 联邦学习概述

  1. 寻求一种方法，训练机器模型 | 不必所有数据集中到一个中心
  2. 单独训练一个模型 | 每个数据源 >>> 各模型彼此交流沟通 | 设计交换模型信息的过程 => 保护隐私 >>> 模型聚合得到全局模型 => 仿佛数据源整合

1.2.1 联邦学习的定义

  1. 联邦学习 = { 模型训练，模型推理 }
  2. 模型训练阶段：模型相关信息共享 | 数据不共享 | 交换过程不会泄露隐私
  3. 模型推理阶段：协作预测 >>> 公平的价值分配机制 => 分配协同模型获得的收益
  4. 特点：
    1. 有多个参与方 | 持有数据
    2. 数据不离开所有者
    3. 模型信息在各方之间传输交换 | 加密 | 无法根据模型推测数据
    4. 性能充分逼近理想模型（集中数据训练的模型）
    5. 中央协调方；可以有也可以不涉及
  5. 挑战：
    1. 通信链路：不稳定；速度慢
    2. 样本数量 or 分布 ：不均衡 => 联邦模型产生偏差
    3. 容易遭到恶意攻击： 破坏性的模型更新信息 => 可用性降低 等

1.2.2 联邦学习的分类

  1. 联邦学习 ={ 横向联邦学习，纵向联邦学习，迁移联邦学习}
  2. 数据组成：样本ID: III，特征空间 XXX，数据标签空间 YYY ， D=(I,X,Y)D=(I,X,Y)D=(I,X,Y)
  3. 横向联邦学习：
    1. 按样本划分的联邦学习
    2. 重叠的数据特征较多
    3. eg：不同地区的银行
    4. 数据特征取交集，数据ID取并集
  4. 纵向联邦学习：
    1. 按特征划分的联邦学习
    2. 重叠的数据较多 ID
    3. eg：一家银行一家电商，客户群体上有很大交集，提供不同服务
    4. 数据取交集，特征取并集
  5. 联邦迁移学习
    1. 重叠都比较少的时候

1.3 联邦学习的发展

1.3.1 联邦学习的研究

  1. Secureboost，提升纵向联邦学习安全性 | 无性能损失 | 隐私保护 | 提升树系统架构
  2. 联邦学习的脆弱性，很容易被投毒
  3. 联邦学习和强化学习
  4. 多任务学习 vs 联邦学习的统计学难题

1.3.2 开源平台

  1. Federated AI Technology Enabler （FATE）微众银行 | 开源项目 | 安全计算框架 | 联邦学习平台
  2. TensorFlow Federated （TFF） 开源框架
  3. TensorFlow-Encrypted 搭建在tensorflow顶层的python包，用于实验 => 机器学习方式|面向隐私保护
  4. coMind 开源平台 | 联邦平均算法的实现 <= 关键组件
    1. 搭建在tensorflow顶层并提供api
  5. Horovod 由Uber创立，基于开放的消息传输接口（Message Passing Interface MPI）
  6. OpenMined/PySyft，提供两种方法 | 隐私保护 { 联邦学习 ， 差分隐私 }

FedAI Ecosystem

2. 隐私、安全及机器学习

介绍背景知识 | 机器学习相关 | 面向隐私保护 { 机器学习技术， 数据分析 }

2.1 面向隐私保护的机器学习

Privac-Preserving Machine Learning，PPML 系统

由个人，团体或机构自行决定何时、如何以及在多大程度上将有关他们的信息传达给他人 ── Alan F.Westin

信息安全的主要思想是控制对个人信息的收集和处理的过程
