俄航出国经验总结

2021-08-10 Life experience 0 Comments Word Count: 2.1k(words) Read Count: 7(minutes)

​ 2021.8.7俄航飞瑞典求学，一路上还算比较顺利地到达了KTH，总结一下经验

1. 订机票

首先提前一两个月订机票，我订的还算早，订到了比较便宜的机票。这次因为疫情，飞国外的话比较麻烦。最赚的是香港飞的，从蛇口码头坐船到香港国际机场，免入境直接起飞。比较便宜，而且时间好，可以赶上8.2接机。

俄航算是比较便宜的，但是时间上不太友好。一个是因为太早了（早上7:10飞），要提前来上海蹲着。一个是8.7有点不早不晚的，白交半个月房租。

订机票的时候还要考虑到航班取消的问题，疫情期间有时候航班会被迫取消，这样可能会打乱之后的计划。不过俄航每周就飞一天，两班飞机，很稳定没取消过。

最后要提前了解入境政策和中转国家过境政策。莫斯科是24小时内免入境转机，但丹麦好像就需要过境签，需要提前准备好。

订机票可以在官网订，也可以在飞猪、携程等平台上订，不过要看清除机票来源，避免被坑（所以果然还是官网订比较稳妥，就是麻烦了点）。

2. 起飞前

2.1 booking code

订完机票之后会收到邮件，在附件里边可以看到自己的booking code，也就是那个六位大写字母的号码，这个之后值机的时候会用到，比较重要。至于航站楼之类的，其实不太需要提前关注，因为有可能会变化，以值机的时候的信息为准。

2.2 关于核酸

  1. 俄航目前的规定是需要72小时的英文核算检测报告。所以要么做好然后赶飞机，要么提前一天来上海做。

  2. 需要提前查一下在哪些位置可以开中英文核算检测报告，以及：

    1. 是否需要预约，在哪里预约
    2. 一周中的哪几天工作，每天的工作时间
    3. 结果多久能出，以及费用问题

    这些信息一般网上能查到，但最好打电话确认一下，像我在之江检测所做的，虽然说要预约，但可以到了现场填信息约，也不麻烦。然后最便宜的标准检测80元，说是12-24小时出结果，但其实一般上午去做了下午4点多就出结果可以去取了，完全没必要加急（当时医生就说了，上午做一般下午就能取），但要注意他们的工作时间。人不多，一般不用排队。

  3. 最后有一点可以确认一下，就是俄航对检测方法有要求，可以确认一下，一般都是常规荧光检测什么的。

取的纸质的报告单一定不要弄丢了，最好和护照放一起。

2.3 住宿

最后综合以上情况，我是选择提前两天到上海，8.7号早上的飞机，我是8.5快中午做的核酸报告。为了方便取报告，选择的住宿的位置也是离检测所的位置比较近的。

因为俄航是一大早从上海浦东国际机场起飞，而机场离市区相当tmd远，如果要在上海玩一下的话，不建议在机场附近住。机场附近的酒店都有送机巴士，但时间一般都是早上4:00之后，比较晚。

所以凌晨三点多去机场，基本上只有打车去一种选择了（机场大厅并不舒服，强烈不建议去那挨一晚上，难受死）

我是使用高德打车，30公里左右吧到机场，大概90元

2.4 值机

俄航提前24小时可以在线值机，主要是选座位。主要用到booking code，提前在知乎之类的地方了解一下选座位怎么选以及哪些座位对长途飞行比较友好（虽然这些座位一般都抢不到或者不让选）。

2.5 机场流程

把自己和一堆行李折腾到机场之后就比较简单了，首先找辆行李车把箱子啊包啥的都往上边一堆（这个车车在机场门外边，找不到问一下，一定要整一辆）。然后推着进机场，要过一道安检。

之后找到值机和行李托运的柜台（因为在线值机过了，PVG的值机和行李托运好像在一起），问一下工作人员俄罗斯航空的值机柜台在哪里，过去排队就行了。

我当时去得有点晚，提前2.5h到的，光在这里就排了tmd1.5h，强烈建议至少提前3h，最好提前3.5h到，免得在那里带一堆行李站半天。

这里会有人查核酸，提前准备好护照、居留卡、核酸证明就行了。等排队排到了就带着大包小包去办理，整个流程不太复杂，按部就班来就行。需要注意：1. 行李不要超重 2. 可以带一个登机箱或者包（10kg），再背一个随身小包（5kg），尺寸查的不严，重量其实也不太严/笑哭，但最好别挑战。

行李最好用那种专门的行李打包带包一下，实在不行用保鲜膜或者胶带包一圈也行，主要防止有人私自打开你的行李塞些baisefenmo什么的。

行李托运办完之后就按部就班地过边检、安检，之后去对应的登机口侯着就行了。

  1. 在办完行李托运之后还可以满机场乱跑，行动不受限，和亲戚道别啥的可以先办完托运再说
  2. 过海关不能拍照
  3. 安检的时候会让你把笔记本、键盘、平板等电子设备都拿出来单独过安检，所以，整理了也白整理（但行李托运好像没这问题，不过行李托运也不能托运这些就是了）。乖乖拿出来过了安检再重新装包里吧

3. 飞行期间

飞行期间没太多需要注意的，一些tips：

  1. 俄罗斯小姐姐真好看
  2. 俄罗斯小姐姐真好看
  3. 俄罗斯小姐姐真好看
  4. 带一双一次性拖鞋飞机上穿，更舒服一点
  5. 包放头顶上，一个小包可以放脚下，不过会压缩本就不大的休息空间，所以建议把证件、手机之类的随身，其他的都扔头顶
  6. 英语交流，加手势，一般都能交流
  7. 第一程飞机会有两餐，所以乘机那天可以不吃早饭
  8. 第二程飞机会有一餐，但由于疫情，最好留着到瑞典了再吃
  9. 第一程吃饭上厕所都OK，第二程就憋住，不吃不喝不拉不撒，戴好口罩和手套（手套好像没人会查，但最好戴吧）等防疫用品，和小姐姐说不要水，thanks

4. 转机

只需要在moscow转一次飞机，比较简单，跟着路牌（有中文）走就行了，过安检也比较简单，不需要憨皮地掏电子产品，但是要摘手表和皮带，注意下别裤子掉了就行。

轶闻：因为摘皮带用了一小会儿，皮带就和下一个妹子的箱子放一个篮子里过安检，然后那妹子的箱子没过，要排队检查，，，导致我提着裤子等了半天拿皮带。。。。。

可以去服务台问一下候机在哪个口，这个有可能会变化。（当然，也有显示屏会显示）

5. 下飞机来KTH

下飞机之后走过长长的通道去领行李，之后去过海关。之后就顺利下飞机了，想办法来KTH就行。

建议方法：express买票，坐车到CentralStation，然后转地铁14号线，坐moroby central方向，到tekniska hogskolan下，直走出站就能看到KTH的校标了。
