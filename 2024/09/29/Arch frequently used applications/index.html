Arch frequently used applications

2024-09-29 Arch Linux applications 0 Comments Word Count: 208(words) Read Count: 1(minutes)

Browser

  • Brave

  • Firefox

  • w3m : TUI browser running in terminal

  • Dolphin: file explorer

Text & Note

  • Nvim
  • Joplin with cloud sync
  • Kate: editor
  • VS Codium: opensource vscode, easy to use
  • heynote: easy draft paper for programmers
  • Libreoffice: opensource alternative to microsoft office
  • Okular: pdf reader
  • Zotero: academic paper manager
  • marktext: markdown editor
  • ImHex: hex editor

Media

  • OBS Studio: screen recorder
  • GIMP: photo editor
  • flameshot: screenshot tool

Tools:

  • rclone: swiss knife for cloud storage including sync, mount, read, write, etc.

  • scc: Cound the statistics of codes in your project

  • onefetch: github repo statistics

  • fastfetch: alternative to neofetch

  • rofi: fast runner

  • keyd: remap keyboard tool

  • exa: advanced ls

  • dust: advanced du

  • duf: advanced df

  • yazi: terminal file manager, graceful and simple

  • docker: run some tools like it-tools or vaultwarden

  • timeshift: System backup and snapshoot manage tool (Especially important for Arch)

  • collectd: collect data of system usage, for performance analyzing

  • vaultwarden: managing credentials and passwords

  • cmd-wrapped: command usage statistics

  • pot: translator

  • safeeyes: break reminder to protect eyes and rest

  • clash-verge: clash proxy client

  • kde-connect: connect my Arch with my phone

  • fcitx5 with rime: pinyin input solution

  • btop: system monitor, advanced alternative of htop

  • bat: alternative to cat

  • Sniffnet: network traffic analyzer and capture

  • ufw or firewalld: Firewall manager
