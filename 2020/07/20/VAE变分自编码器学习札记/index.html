变分自编码器（VAE）学习札记

2020-07-20 ML ML, VAE, python 0 Comments Word Count: 2.3k(words) Read Count: 8(minutes)

变分自编码器（VAE）是一种很主流的生成模型，它可以通过分析原始数据，并提取特征，将特征映射到z空间，再取样解码生成新的数据。生成的数据和原始的数据略有不同但相似。VAE以其精妙的数学原理为人称道，被广泛应用于数据增强、图像生成等领域。

1. VAE简介

简单来讲，一个可以和GAN相媲美的生成网络。我们可以输入 n 维空间的Z，映射到 k 维空间的真实数据。比如，生成不同样的数字，人脸，卡通头像等等。

上图就具体展示了VAE的作用，左图是经过一轮训练之后的输出结果，中间的图是经过9轮训练之后的输出结果，而右图就是我们真实地数据。（这里是编码解码一体，也就是编码真实数据，然后再解码回真实数据，所以长相差不多。实际使用的时候是随机生成z，然后喂入解码器生成数据，不需要编码器）

https://blog.csdn.net/weixin_40955254/article/details/82315224

2. VAE原理

此图为VAE的全部结构图，训练完毕后实际使用的只有decoder部分，也就是自定义z，然后解码生成x‘

2.1 VAE思路

为了生成新的图像，那么一个很自然的思路就是提取原图像的特征。也就是建立图像空间 X 和特征空间 Z 之间的双向映射关系。然后我们直接从 Z 空间中取样 z，再经过decoder就得到了 X 空间的图像 x’ 。只要我们的映射关系足够好，那么我们生成的图像就可以很逼真。接下来要解决的问题就是：

  1. Z 空间如何构造？总不可能人为指定吧
  2. X Z 的映射关系用什么表示？
  3. x样本有限，有限的x如何求出”无限“的映射关系？

总结一下就是：如何构建 X 到 Z 的映射？

原论文中其实是求联合概率分布，那种思路有点太数学了，用到了很多概率论的知识。我不确定我现在理解了之后还能不能回忆起来，反正是当个工具用嘛，就提供另一种简单的理解方式，空间映射。

2.2 构造方式

2.2.1 问题1、2：Z空间如何构造？X Z 的映射关系如何表示？

似乎有句话是这么说的：“不好拟合的东西全部用神经网络去做”。所以，神经网络毫无疑问可以用来做encoder和decoder。简单一点说，就是用神经网络去学习特征，我们只需要指出 Z 的维度，然后就用神经网络去拟合 X 到 Z 的encoder就好，同样，也用神经网络去拟合 Z 到 X 的decoder。那么，只要我们以某种方式去训练神经网络，就可以达到用两个神经网络当做映射器的目的。两个问题都迎刃而解了，用神经网络当做映射器，人为指定 Z 的维度就好，至于 Z 的维度到底代表什么含义，怎么算的，那就只有神经网络知道了。我们只关心X Z 可以以某种方式关联映射就好。那么接下来的问题就变为：

  1. 神经网络的结构如何？如何训练？

  2. x 样本有限，有限的 x 如何求出“无限”的映射关系？

2.2.2 问题3：x 样本有限，有限的x如何求出“无限”的映射关系？

在训练神经网络之前，我们要先来决定我们的映射策略。x的样本总是有限的，但 X 图像空间是无限的，包含无限的图片。而我们的目的正是通过有限的 x ，去生成更多的 x‘ ，x’ 也包含在 X 空间内。那么，该如何做到呢？如果只是简单的x->z->x的方式来训练，那么最好的结果，就是可以单一的求出所有的样本 x 在 Z 空间的单一映射，那么取特定的 z 反映射可以得到特定的 x （样本）。那么在其他的地方取 z 再映射回去得到什么，那就只有天知道了。所以，为了实现从 X 取样求出所有的映射，我们在 z 中混入高斯噪声，也就是：x->z->z’->x’，其中z->z’ 就是在 z 的附近取一个 z’ 再映射回 x’，这样的话就可以保证映射回去的 x‘ 不会完全和 x 相同。那么接下来又有问题了：

  1. 如何保证在 z 附近取得的 z’ 映射回去的 x’ 和 x相似呢？也就是说如何保证 Z 空间中相近的地方在 X 空间中也相近？
  2. 如何保证 X 的特征映射到 Z 之后的相关程度就符合高斯分布？也就是说，凭什么要用高斯噪声呢？可想而知，要使 X 空间映射到 Z 之后，z’ 和 z_0 之间的相似度符合高斯分布，那么训练encoder的时候一定需要某个条件约束。（简单点理解就是横轴是z’ 和 z0 的距离，纵轴是 x和x‘ 的相似程度，绘制出来的图像符合高斯分布），那么这个约束条件是什么？

这里原论文是使用的条件概率，我这里抽象成了z->z’的取样，实际上如何是用的条件概率，那么就不需要这一步，也就是说，X 到 Z 并不是一一映射，而是服从条件概率，x只是最可能映射到z，也可能映射到 z 附近，而为了保证这个条件概率是服从高斯分布的，需要进行数学证明推导出一个encoder 的约束条件，稍后会给出

2.2.3 问题4： 神经网络的结构如何？如何训练？

还记得么？z -> z’ 是需要进行一次变换的，我们可以认为 z 就是 x 的特征向量，那么如何取 z‘ ？ 就需要均值方差啦~均值就是 x 的特征向量， 方差其实就是高斯分布的方差咯，而这个映射实际上可以表示成 z’ = 均值 + 方差*标准高斯分布，即： 这样就得到 z’ 啦（这个过程也叫做重参数化，不过，whatever，不重要）

如何训练神经网络？这个问题也就一并解决了问题5和问题6。如何训练神经网络，其实就是如何构造loss函数的问题，一旦loss函数构造完成了，那么只需要最小化loss即可。

回顾一下我们神经网络的目的：拟合X和Z的空间映射关系，并满足相对关联性（x，x‘ 相近，z，z’也应该相近）
那么，将目的更加具体化：VAE的训练流程是：x->z->z’->x’，那么，我们要：

  1. 让 x 和 x’ 尽量像
  2. 和x长得像的所有x‘到Z的映射z’都要在z周围，假设表征 x 和 x’ 的相似度，那么 近似服从高斯分布
